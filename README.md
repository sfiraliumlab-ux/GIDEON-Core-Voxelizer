## 🌀 Реноме Проекта

**GIDEON Core Voxelizer** — это экспериментальный программный комплекс, реализующий альтернативную парадигму адресации и пространственного распределения данных. Приложение преобразует плоские двухмерные массивы информации (растровые изображения, кадры сенсоров и видеопотоки) в трехмерные облака вокселей, нанизанные на строго детерминированную геометрическую ось — **Сфираль**.

В отличие от классических систем ИИ, использующих линейные тензорные матрицы и ресурсоемкие механизмы внимания (Attention), архитектура GIDEON осуществляет **аппаратное сжатие контекста** за счет топологических свойств золотого сечения (числа $\Phi$).

### 🧬 Математический Базис Топологии

Геометрия структуры жестко перенесена из оригинальных R&D-скриптов автора (О.С. Басаргин) для среды Blender и состоит из двух зеркально антисимметричных фаз, объединенных оператором инверсии:

1. **S-образная петля инверсии (Центральный узел):** Параметрический полукруг, диаметр которого равен радиусу основного витка ($R_{coil}$), со смещением геометрического центра в точку $R/2$. В области перехода ($t=0$) реализуется фазовая инверсия знака потенциала.
2. **Основной виток:** Сбалансированная окружность постоянного радиуса $R_{coil}$, бесшовно подхватывающая траекторию S-дуги и формирующая замкнутый цикл контекста (ровно $360^\circ$ на каждую половину).
3. **Фундаментальная Антисимметрия:** Левый и правый витки связаны строгим оператором зеркальности по всем трем пространственно-временным координатам: $P_{left} = -P_{right}$.

## 🛠 Текущий Функционал Среды

- **Синтетический Паттерн:** Математическая генерация канонического градиентного каркаса Сфирали с визуализацией троичной логики `{-1, 0, 1}` (переход от Синего фазового потенциала торможения к Красному потенциалу возбуждения).
- **Импорт Растровых Кадров:** Попиксельная декомпозиция сторонних изображений и их ортогональный маппинг на 3D-ось.
- **Интеграция с Сенсорами:** Захват данных с веб-камеры через изолированный защищенный буфер для обхода CORS-ограничений WebAPI.
- **Интерактивный 3D-Проектор:** Свободное ручное манипулирование и вращение структуры в пространстве (Drag-to-Rotate) для анализа глубины S-моста.
- **Экспорт Матрицы (3D R&D):** Моментальная выгрузка полученного облака вокселей со спектральными RGB-данными в стандартный формат **Wavefront .OBJ** для последующей сборки в Blender.

## 🚀 Перспективы Развития Архитектуры

Проект GIDEON закладывает основу для преодоления кризиса энергоэффективности современных ИИ-чипов ("бутылочного горлышка" фон Неймана). Векторы дальнейшего развития кодовой базы включают:

1. **Аппаратные Шейдеры (WebGL GPU Stream):** Перенос математического ядра на язык GLSL для потоковой обработки видео со скоростью 60+ FPS без прерывания на снимки буфера.
2. **Нейроморфные Сфиральные Сети (FSIN):** Замена классических функций активации нейронов (ReLU, Sigmoid) на непрерывный фазовый мост S-петли.
3. **Автоматизация Сборки в Blender:** Создание Python-аддона, автоматически преобразующего экспортированные .OBJ вершины в сглаженные меш-проводники.
